  • No pycharm criaremos um novo projeto. A vantagem do paycharm é que quando se cria o projeto automaticamente o ambiente virtual também é criado.
  • O projeto será monitorado pelo GIT. Dentro do diretório do projeto digite os camandos do GIT
    • Os comandos são encontrados aqui
      • 1º - git init - Cria um novo subdiretório chamado .git que contém todos os seus arquivos de repositório necessários
      • 2º - git add - Adiciona os arquivos a serem monitorados pelo git
      • 3º - git commit - salva o arquivo no estado atual
      • Todos os comandos pode ser visto passo a passo na documentação. Navegue por ela. CLIQUE AQUI
      • Após devemos submeter nosso código ao github CLIQUE AQUI PARA DETALHES

INICIANDO O PROJETO

  • No pycharm vá em File-New Project . Coloque o nome do projeto CadastroBeneficio. Clique em criar novo projeto
  • Será criado o diretorio com o nome do projeto e dentro do diretorio o ambiente virtual. Ative o ambiente virtual digitando: venv\Scripts\activate .
  • Vamos usar o git para monitorar o nosso projeto.

    Digite:

    git init - cria o arquivo oculto .git com todas as funcionalidades do git

    git configure user.name "Orlando Pedro"

    git configure user.name "opsoutoferreira@gmail.com"

  • Para verificar se as configurações foram realizadas entre no diretório .git e digite:

    TYPE configure

    Volte para o diretório do projeto e digite git status

    Aparecerá os arquivos para adicionar ao monitoramento. Vamos monitorar somente os arquivos que trabalharemos. Os arquivos criados pelo git não

    Para ignorar os arquivos que não serão monitorados devemos adicioná-los no gitignore. Para tanto crie o arquivo .gitignore

    Adicione os arquivos ou diretórios que serão ignorados

    Agora podemos adicionar os arquivos a serem monitorados e depois comita-los
